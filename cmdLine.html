Linux基础命令

最近刚学了一些Linux基础命令，在这里分享给朋友们。 （备注：我这是在window7系统上安装的Gitbash上模拟实现的Linux命令）

目录（文件夹）操作

  • 命令1：（进入桌面） 代码: cd ~/Desktop
  • 命令2：（创建目录demo_1（文件夹）） 代码：mkdir demo_1
  • 命令3：（创建多个目录（文件夹）） 代码：mkdir -p demo_2/demo_3
  • 备注：如果文件夹名中含特殊字符和空格，需要加引号，代码：mkdir "demo 1"
  • 命令4：（创建多个目录（文件夹）） 代码：mkdir -p demo_2/demo_3
  • 命令5：（删除桌面下目录demo_2/demo_3（文件夹）） 代码：rm -rf demo_2
  • 备注：如果你此时位置就在桌面可以直接用代码： cd demo_1
  • 命令6：（返回上级目录） 代码： cd ../
  • 命令7：（进入目录demo_1/demo_2（文件夹）） 代码：cd ~/Desktop

文件操作

  • 命令1：（进入桌面） 代码: cd ~/Desktop
  • 命令2：（创建一个index.txt文件） 代码: touch index.txt
  • 命令3：（打开一个index.txt文件） 代码: start index.txt
  • 备注：此时你进入了index.txt文件，可以进行编辑。例如写"hello world"然后保存退出TXT文件
  • 命令3：（查看index.txt文件内容） 代码: cat index.txt
  • 备注：此时你发现index.txt文件内容"hello world"显示在命令行中。但如果你输入是中文，会出现乱码！！
  • 命令4：（重命名index.txt文件为1.txt） 代码: mv index.txt 1.txt
  • 命令5：（查看1.txt文件内容） 代码: cat 1.txt
  • 命令6：（删除1.txt文件内容） 代码: rm 1.txt

查看文件路径

  • 命令1：（进入桌面） 代码: cd ~/Desktop
  • 命令2：（查看桌面上所有目录和文件(不包括隐藏文件和下级文件)） 代码: ls
  • 命令3：（查看桌面上所有目录和文件(包括隐藏文件和下级文件)） 代码: ls -a
  • 备注：隐藏文件为文件名前带"."的文件，你可以创建一个".1.txt"文件，你会发现它在桌面上被隐藏掉了！！ 但用命令ls -a可以查看到
  • 命令4：（查看桌面上所有目录和文件(不包括隐藏文件和下级文件)）且会显示更多文件信息 代码: ls -l
  • 备注：“drwxr-xr-x” 中 "d"表示directory是一个文件 "r"表示可阅读； "w"表示可书写；"x"表示可执行；
  • 命令5：（查看桌面上所有目录和文件(包括隐藏文件和下级文件)）且会显示更多文件信息 代码: ls -l
  • 命令5：（查看1.txt文件内容） 代码: cat 1.txt
  • 命令6：（删除1.txt文件内容） 代码: rm 1.txt
